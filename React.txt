[리액트란?]
- 사용자 인터페이스를 만들기 위한 자바스크립트 라이브러리
- SPA(Sing Page Application)를 쉽고 빠르게 만들 수 있도록 해 주는 도구
- 빠른 업데이트와 렌더링 속도, 재사용성이 높은 컴포넌트 기반 구조
- 메타의 든든한 지원, 활발한 지식 공유와 커뮤니티
- 리액트 네이티브를 통한 모바일 앱 개발 가능

- 방대한 학습량(https://react.dev)

[JSX]
- 자바스크립트와 HTML 태그를 섞어 사용하는 문법
  자바스크립트 표현식은 중괄호{}로 표현
  자바스크립트 표현식을 HTML 태그와 함께 사용할 수 있어 가독성 있는 코드를 작성할 수 있음.
  숫자, 문자열, 불리언, Null, undefined, 배열 사용 가능
  객체는 반드시 속성으로 표기해야 함. objA.b

- 작성 규칙
  모든 태그는 닫는 태그 </>반드시 필요 : <img />, <input />
  반환하는 모든 태그는 반드시 최상위 태그로 감싸줘야 함.
  만약 마땅한 HTML 태그가 없다면.... 빈 태그 <> </> 사용 가능함.
